JavaScript FAQ's


What is JavaScript?

JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and much more.

What is jQuery?

jQuery is the library of the JavaScript fucntions that you're most likely to need as you develop web sites.

Why is jQuery becoming so popular?

With over 10 years of development, jQuery has accumulated a vast array of useful plugins, making it an attractive choice for developers who need to quickly implement common functionality.
